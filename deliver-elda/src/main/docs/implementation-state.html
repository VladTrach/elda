Elda -- an implementation of the Linked Data API

state of the implementation

exists

exists and not-exists are not implemented at present.

views

The explicit describeViewer, labelledDescribeViewer, basicViewer are not currently available.

formatting

The HTML renderer is very primitive and there is no XSLT support.

Not all of the detailed language-tag and literal-type features work as specified.

cache

Elda caches the results of queries so that they may be re-served quickly. At the moment these caches are never emptied until the Elda server exits. This has two consequences: the server may unexpectedly run out of room, and changes to the remote dataset may not be reflected in queries using the API.

HTML and XML rendering

The current HTML renderer is very primitive and there is no XML renderer.
