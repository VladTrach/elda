Interoperability issues

It's easy to write simple URI queries that generate surprisingly complicated SPARQL.

  • Property chains especially when combined with big data windows (200K SPARQL queries!).
  • Some endpoints do less optimisation than others. Elda does a primitive literals-to-the-top reordering as a work-around.
